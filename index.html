Smart Data. Smarter Decisions.

Smartmove is an open source data platform that aims to provide businesses and consumers with insight into the property market in Ireland and the UK.

Features

Data Warehouse

By utilizing data from multiple sources including Daft.ie, Zoopla and the Property Price Register, the project features a Data Warehouse which allows for fast reporting of aggregated statistics.

RESTful API

Smartmove includes a REST API that allows users to use Smartmove data in their own applications. The API includes features such as the ability to load JSON data that can easily be consumed by the Google Charts API.

Client Libraries

Smartmove is currently developing client libraries that will allow you to easily interact with the API and make data driven decisions programmatically.

Smartmove empowers your business decisions through relevant data about specific locations.
